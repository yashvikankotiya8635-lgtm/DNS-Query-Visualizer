# DNS Query Visualizer

A Python-based cybersecurity project that captures and visualizes DNS traffic using Scapy, Flask, and Folium.

## Features
- DNS packet sniffing
- GeoIP-based location tracking
- Interactive DNS world map
- Top queried domains graph
- Flask web dashboard

## Technologies Used
- Python
- Scapy
- Flask
- Folium
- Pandas
- Matplotlib

## How to Run

1. Install requirements
2. Run app.py
3. Open localhost server

## Project Type
Cybersecurity + Network Monitoring
